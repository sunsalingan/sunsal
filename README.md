# RankEats (순살 랭킹)

RankEats는 사용자가 직접 만드는 신뢰할 수 있는 맛집 랭킹 플랫폼입니다. 내가 가본 맛집들의 순위를 매기고, 친구들과 공유하며 진짜 맛집을 발견하세요.

## 🚀 프로젝트 시작하기

### 1. 필수 요구사항
- Node.js (v16 이상 권장)
- npm

### 2. 설치 및 실행
프로젝트 루트 디렉토리에서 다음 명령어를 실행하세요.

```bash
# 의존성 패키지 설치
npm install

# 개발 서버 실행
npm run dev
```

### 3. 주요 기능
- **맛집 랭킹**: 나만의 맛집 리스트를 생성하고 순위를 매길 수 있습니다.
- **점수 시스템 고도화**:
    - **가중치 점수**: 리뷰를 많이 작성한 유저의 점수가 더 크게 반영됩니다.
    - **랭킹 모순 방지**: 랭킹 설정 시 논리적 오류를 방지하는 가이드 시스템.
- **친구 랭킹 공유**: 친구의 랭킹과 내 랭킹을 비교하고, 취향(Match Rate)을 확인할 수 있습니다.
- **지도 고도화 (Naver Maps)**:
    - **지인 지도**: 나뿐만 아니라 지인들의 맛집 랭킹을 지도 위에 통합 표시합니다.
    - **마커 클러스터링**: 줌 레벨에 따라 인접한 마커들을 그룹화하고, 대표 마커(최고 순위)와 '+N' 배지로 표시합니다.
    - **상세 필터링**: 카테고리(아시안, 한식, 일식 등)별 필터링 및 지도 중심 기반 실시간 랭킹 리스트 동기화.
- **리뷰 작성**: 위치 인증 및 영수증 인증(프로토타입) 절차를 거친 신뢰도 높은 리뷰 시스템.
- **관리자 기능**: 전체 데이터 백업(Export) 및 복구(Import), 초기화 기능.

## 🛠️ 기술 스택
- **Library**: React, Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Map**: Naver Maps API
- **Database**: Firebase (Firestore, Auth)

## 📂 프로젝트 구조
```
src/
├── components/
│   ├── features/       # 주요 기능 컴포넌트 (MapArea, ReviewModal, Sidebar 등)
│   ├── layout/         # 레이아웃 컴포넌트 (Header 등)
│   └── ui/             # 공통 UI 요소
├── services/           # Naver Search API 등 외부 API 통신 레이어
├── data/               # Mock 데이터 및 상수
├── lib/                # 외부 라이브러리 설정 (Firebase 등)
├── App.jsx             # 메인 앱 컴포넌트
└── main.jsx            # 진입점
```
