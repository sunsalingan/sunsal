# 🎯 프로젝트 목표 및 로드맵 (Target Summary)

## 🌟 프로젝트 비전 (Vision)
**RankEats (Sunsal)**: 사용자 중심의 신뢰할 수 있는 맛집 랭킹 및 지도 서비스.
단순 평점이 아닌, 신뢰도 높은 리뷰 시스템과 유저 간 인터랙션(친구/커뮤니티)을 통해 개인화된 맛집 탐색 경험을 제공합니다.

## 📍 현재 단계 (Current Status)
**Version 1.9** - 랭킹 로직 고도화 및 기반 시스템 안정화
-   [x] **랭킹 알고리즘**: 가중치 기반 동점자 처리 및 점수 산정 로직 개선
-   [x] **관리자 기능**: 데이터 초기화/백업/복구 및 유저 닉네임 관리 기능
-   [x] **지도 연동**: 네이버 지도 API 기반 마커 클러스터링 및 필터링

## 🥅 단기 목표 (Short-term Targets)
### 1. 커뮤니티 및 소셜 기능 (Next Priority)
-   [ ] **친구 시스템 고도화**: 유저 검색, 팔로우/언팔로우 기능 완성 및 UI 개선
-   [ ] **소셜 피드**: 팔로우한 친구들의 리뷰와 활동을 모아보는 피드 구축

### 2. 사용자 경험(UX) 및 인터페이스 개선
-   [ ] **UI/UX 폴리싱**: 다크 모드(Dark Mode) 지원 및 모바일 반응형 디자인 최적화
-   [ ] **편의성 개선**: 검색 기능 강화, 직관적인 네비게이션 및 온보딩 추가

## 🚀 장기 로드맵 (Long-term Roadmap)
### 1. 플랫폼 확장 및 인증
-   [ ] **소셜 로그인**: 카카오/구글 연동을 통한 간편 가입 및 프로필 연동
-   [ ] **PWA (Progressive Web App)**: 앱 수준의 사용자 경험 제공 및 오프라인 접근성 확보

### 2. 데이터 기반 개인화
-   [ ] **개인화 추천 알고리즘**: 유저 활동/취향 분석을 통한 맞춤형 맛집 추천
-   [ ] **스마트 리뷰 분석**: AI를 활용한 리뷰 키워드 추출 및 요약 정보 제공
