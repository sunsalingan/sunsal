## 🐛 알려진 문제 (Known Issues)
- **일치도 변경 현상**: 팔로우/언팔로우 동작 수행 시 화면이 리렌더링되면서 `Math.random()`으로 설정된 취향 일치도 값이 변하는 현상이 있음. (추후 실제 데이터 기반 알고리즘으로 교체 시 해결 예정).
- **위치 기반 검색 정확도**: 네이버 Place API 검색 결과가 "현재 지도 위치"보다 "텍스트 연관성"에 너무 치우치는 경향이 있어, "내 주변 롯데리아" 등을 찾을 때 먼 곳이 나오기도 함.
